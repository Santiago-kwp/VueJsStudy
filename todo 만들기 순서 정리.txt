할 일 목록 입력받기
1. 데이터를 정의하고, v-model 디렉티브로 입력창과 양방향 데이터 바인딩
- input 태그에 v-model="newTodo"
- 입력한 값을 확인하는 이벤트 등록 => methods에 addTodo 작성
- emit 등록해서 App.vue에 데이터 전달 및 전달 후 초기화
  - $emit() 함수를 사용해 자식 컴포넌트에서 부모 컴포넌트로 데이터를 전달하도록 코드 작성
  - emits 옵션 속성과 $emit 함수를 사용해 발신하는 코드 작성

2. App.vue에서 전달받은 todo를 처리하기
- App.vue의 메소드에 addTodo 함수를 정의해서 App.vue에 todos에 TodoInput.vue에서 넘어온 데이터 추가

3. 할 일 목록 필터링하기
- 필터 데이터 정의 (filter: "all")
- 정의한 filter 데이터를 TodoFilter 컴포넌트에 전달 (:filter="filter")
- TodoFilter.vue에서 props를 등록하여 해당 filter값을 받음
- filter 데이터 값을 사용해 현재 활성화 상태 정의(각 상태를 클릭하면 전달받은 filter 데이터의 값도 변경해야 함)
- App.vue에 update-tab 이벤트 정의하고 등록=> <TodoFilter @update-tab="updateTab">
- App.vue에 computed 옵션 속성 작성 => switch - case 문 구분해서

4. 할 일 목록 출력하기
- App.vue에서 computedTodo 데이터를 TodoList 컴포넌트로 전달
- 부모 컴포넌트에서 전달한 computedTodo 데이터를 TodoList 컴포넌트에서 props 옵션 속성으로 받기
- 전달받은 computedTodo 데이터로 할 일 목록을 출력
- computedTodo를 반복 순회하면서 TodoItem.vue에 item으로 props 전달
